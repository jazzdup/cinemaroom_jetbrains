This problem is fully supported by version 2021.9 of the EduTools plugin and newer. To avoid possible errors, please make sure that the plugin is updated. Alternatively, you can solve it using the web interface.

Currently the program stores and returns items. Extend the program and add the possibility to delete items. Here are the details:

DELETE /items/{id}: removes and returns an item with a specified id. If there is no such item, the program returns Item wasn't found. Status code should be 200 (OK).

Do not change the provided template. Just add new functionality to the code.
